## FlipCards app

Simple flip cards app for memorization (e.g. vocabulary practice). For practice, the app shows one sde of the card and the user has to write the other side exactly (excluding provides notes).

The app uses a SM2 variant algorithm to determine when cards are due for repetition.

### Features

- Cards can be organized into different lessons
- Cards can be studied by lesson, or from all due cards
- Optional notes can be provided on each side of a card for additional information
- Interface is currently only available in English
- Dark and light mode
- Inverted study option (showing the back side of the card and asking for the front side)
